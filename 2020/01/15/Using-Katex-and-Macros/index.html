Using Katex and Macros

Posted on 2020-01-15

Katex 是Kahn Academy开发的，比MathJax似乎要快一点的网页数学显示程序库。我也是闲的蛋疼，要在自己的online scratch pad上(typecho)，安装了一下这个东西。

安装很简单，只用在HTML文件的header里加上引用的那几行就好了，跟hypothes.is的安装一样。不用下载什么东西。去官网上看就好了。

使用macro。就像在Latex preamble里，你想写 

\def\P{\mathbb P}

那么只用在文章某地写上

$$\gdef\P{\mathbb{P}}$$

就好了.

另外typecho的插件我用的是bLue的AutoLatex，但是其实，也许直接改template php file更方便。whatever
